## ✨ Overview

**METEO** is a high-performance atmospheric visualization engine that recreates weather conditions from some of the world's most iconic climates.

Designed with a futuristic dashboard interface, the application delivers immersive environmental simulations including realistic rain, snowfall, dynamic particle systems, ambient lighting, and interactive weather telemetry.

Everything is built using **vanilla HTML, CSS, and JavaScript** without external frameworks.

- 🌧️ 100+ Rain Environment Simulations
